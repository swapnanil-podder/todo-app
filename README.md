# Todo App

This repository is a simple starting point for a browser-based todo app.

Right now, the project is intentionally minimal: the core files are in place, but the app logic and UI have not been built out yet. Think of this as a clean desk before the real work starts.

We are doing this as a hobby project with my Friend.
